Brainrest Command-line Tool Overview

Brainrest is a terminal-based micro-break reminder tool that helps prevent eye strain and promote healthy work habits. It notifies you after a chosen interval and locks your terminal for a short 2-minute break using a clean visual countdown interface.

Description

brainrest is a terminal-based break reminder tool that enforces regular micro-breaks. When a break begins, it locks your terminal for 2 minutes with a visual countdown, blocking commands and encouraging you to rest.

The tool includes a centered TUI mode with mouse/keyboard support, real-time countdowns, and an adaptive bordered interface.

Options

-s, --set MINUTES

Set the break interval (default 40 minutes). The value is saved across sessions.

-r, --reset

Reset the break timer to 40 minutes and clear history.

-V, --visual

Launch interactive visual TUI mode with countdowns, color indicators, start/stop controls, configuration, and navigation support:

  • Arrow keys ↑/↓ or j/k
  • Number keys 1–4
  • Mouse clicks
  • ENTER/SPACE to select
  • ESC or ‘q’ to exit

-v, --version

Show version and OS information.

-h, --help

Display the help message with available commands.

Linux Installation

Add the Brainrest repository:

echo "deb [trusted=yes] https://siripireddygiri.github.io/Brainrestall/repo stable main" | sudo tee /etc/apt/sources.list.d/brainrest.list

Update and install:

sudo apt update && sudo apt install brainrest

Reference - Linux

macOS Installation

Install Homebrew if needed:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Tap the Brainrest repository and install:

brew tap siripireddygiri/brainrest
brew install brainrest
