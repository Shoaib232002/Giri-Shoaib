Command-line Tool Overview

  • Command-line tool built for Linux and macOS
  • Helps users take timely breaks by reminding them after working for a set duration
  • For Linux: echo "deb [trusted=yes] https://siripireddygiri.github.io/Brainrestall/repo stable main" | sudo tee /etc/apt/sources.list.d/brainrest.list
  • sudo apt update && sudo apt install brainrest
  • Reference - Linux
  • For macOS: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  • brew tap siripireddygiri/brainrest && brew install brainrest
  • Reference - macOS
