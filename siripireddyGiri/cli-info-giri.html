Brainrest Command-line Tool Overview

Brainrest is a command-line utility designed for Linux and macOS users. It helps you take timely breaks by reminding you after working for a set duration.

Linux Installation

Add the Brainrest repository:

echo "deb [trusted=yes] https://siripireddygiri.github.io/Brainrestall/repo stable main" | sudo tee /etc/apt/sources.list.d/brainrest.list

Update your package list and install Brainrest:

sudo apt update && sudo apt install brainrest

Reference - Linux

macOS Installation

Install Homebrew if it is not already installed:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Tap the Brainrest repository and install the tool:

brew tap siripireddygiri/brainrest
brew install brainrest

Reference - macOS
