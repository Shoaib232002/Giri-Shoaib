Timezones Command-line Tool Overview

Timezones is a simple command-line tool for Linux and macOS that lets users quickly check the current date and time in different cities around the world. It also includes a visual mode that displays live, continuously updating time.

Description

The timezones command prints the current date, time, and timezone information for selected countries or cities. It supports a live visual mode for real-time updates.

Command

timezones [OPTIONS]
timezones -vi     # Enable visual live-time mode

Options

-in — Show India Time (Asia/Kolkata)

-uk — Show United Kingdom Time (Europe/London)

-hk — Show Hong Kong Time (Asia/Hong_Kong)

-d — Show Dubai Time (Asia/Dubai)

-jp — Show Japan Time (Asia/Tokyo)

-mel — Show Melbourne Time (Australia/Melbourne)

-vi — Launch live visual mode with real-time clock updates

Linux Installation

Add the Timezones APT repository:

echo "deb [trusted=yes] https://Shoaib232002.github.io/timezones /" | sudo tee /etc/apt/sources.list.d/timezones.list

Update and install Timezones:

sudo apt update && sudo apt install timezones

macOS Installation
