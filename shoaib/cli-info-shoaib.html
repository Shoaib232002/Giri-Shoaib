Command-line Tool Overview

  • Command-line tool built for Linux and macOS
  • Helps users to check different timezones based on cities or countries
  • For Linux: echo "deb [trusted=yes] https://Shoaib232002.github.io/timezones /" | sudo tee /etc/apt/sources.list.d/timezones.list
  • sudo apt update && sudo apt install timezones
  • Reference - Linux
  • For macOS:
  • brew tap Shoaib232002 && brew install timezones
