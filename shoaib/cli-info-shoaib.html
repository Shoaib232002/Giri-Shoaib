Timezones Command-line Tool Overview

Timezones is a command-line utility for Linux and macOS that allows users to check the current time in different cities or countries.

Command: timezones [OPTIONS], timezones -vi
Description:The timezones command prints time date and timezones based on cities timzones. also available with visual mode which displays live time.
Options: -in Show India time (Asia/Kolkata) -uk Show UK time (Europe/London) -hk Show Hong Kong time -d Show Dubai time -jp Show Japan time -mel Show Melbourne time

Linux Installation

Add the Timezones repository:

echo "deb [trusted=yes] https://Shoaib232002.github.io/timezones /" | sudo tee /etc/apt/sources.list.d/timezones.list

Update your package list and install Timezones:

sudo apt update && sudo apt install timezones

macOS Installation
