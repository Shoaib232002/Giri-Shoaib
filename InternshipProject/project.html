Project Overview

Animation-Arcade is a web project for building interactive animations. It emphasizes modularity, maintainability, and high-quality code for future scalability.

Key Features

  • Modern JavaScript (ES6+)
  • Clean, structured CSS with linting
  • Automated code quality checks
  • Unit testing using Jest
  • Continuous Integration with GitHub Actions

Technology Stack

JavaScript (ES6+)
Core animation logic
CSS
Responsive styling
Node.js (v24.11.1)
Runtime environment
ESLint & Stylelint
Code quality and consistency
Husky & Commitlint
Automated pre-commit checks
Jest
Unit testing framework
GitHub Actions
Continuous integration workflow

Project Setup

  1. Clone repository: git clone <repo-url>
  2. Enter folder: cd Animation-Arcade
  3. Install dependencies: npm install
  4. Use correct Node version: nvm use (v24.11.1)

Code Quality Automation

  • ESLint: JS linting rules and best practices
  • Stylelint: CSS linting and conventions
  • Husky: Pre-commit hooks for linting & tests
  • Commitlint: Enforces Conventional Commits
